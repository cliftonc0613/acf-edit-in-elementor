== Description ==

Allows you to edit ACF fields from the Elementor Page Settings tab.

== Installation ==

Installing "ACF Edit in Elementor" can be done either by searching for "ACF Edit in Elementor" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress
